# @grandgular/rive-angular (Nx workspace)

Nx workspace for building **`@grandgular/rive-angular`** â€” an Angular wrapper around the official Rive JS runtime (similar in spirit to `rive-react`, but for Angular).

## Projects

- **`showcase`**: Angular app for docs + showcase (`apps/showcase`)
- **`@grandgular/rive-angular`**: publishable Angular library (`libs/rive-angular`)

## Common commands

```bash
# Serve docs/showcase app
npx nx serve showcase

# Build the Angular library (APF)
npx nx build rive-angular

# Build the showcase app
npx nx build showcase
```
